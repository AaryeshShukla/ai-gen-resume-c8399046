# Aaryesh Shukla - Resume

This is a premium interactive CV/resume for Aaryesh Shukla, showcasing education, skills, projects, certifications, research work, and a contact form.

## Features
- Ultra-clean grid layout with high-contrast typography.
- Subtle progress bars for skills.
- Clickable projects list that opens in new tabs.
- Responsive design with modern aesthetics.
- Styled contact widget for inquiries.

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts (Inter)

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the resume.
